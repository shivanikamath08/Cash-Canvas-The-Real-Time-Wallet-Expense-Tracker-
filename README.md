CashCanvas is a full-stack web application that helps users efficiently manage personal or household expenses. Users can add, view, and delete expenses, and the application automatically calculates the total expenditure in real-time. Built with HTML, CSS, and JavaScript on the frontend and Node.js, Express, and MongoDB on the backend, it demonstrates CRUD operations, persistent data storage, and dynamic frontend-backend interaction.

Features:

Add new expenses with title, amount, category, and date

Display all expenses dynamically in reverse chronological order

Calculate and update total expenses automatically

Delete expenses easily with a single click

Data is persistently stored in MongoDB

Responsive and user-friendly interface

Tech Stack:

Frontend: HTML, CSS, JavaScript

Backend: Node.js, Express.js

Database: MongoDB
